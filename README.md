<!-- GETTING STARTED -->
## Getting Started
![](assets/4.png)

The UVerifer system is a two part application consisting of :
- Aadhar Authenticator
- POC IRCTC App

### Installation

- Clone the repo using 
```sh
git clone https://github.com/TheSpeedX/UIDAI-Hackathon.git
```

- Web
  - Aadhar Authenticator
  ```sh
  pip install -r requirements.txt
  uvicorn main:app 
  ```
  - POC IRCTC APP API
  ```sh
  npm install
  npm start
  ```
- Android
  - UVerifier
  - POC IRCTC APP
  - Verifier Admin
